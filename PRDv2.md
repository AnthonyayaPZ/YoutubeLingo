# 产品需求文档 (PRD)：本地 YouTube 影子跟读视频生成引擎 (ShadowGen-Local)

## 1. 产品概述
* **产品形态**：基于 Python 的本地命令行自动化工具（CLI）
* **核心目标**：全自动处理 YouTube 视频，生成带硬字幕的“影子跟读”视频及学习素材
* **产品价值**：通过“原音 -> 画面定格+中文释义 -> 原音”模式，解决英语学习者寻找高质量沉浸式跟读素材的痛点

## 2. 核心执行流程 (User Flow)
* **用户触发**：通过终端命令执行（如 `python main.py --url <URL> --target_lang zh`）
* **静默流水线**：
    1.  **抓取**：下载最佳音视频源流
    2.  **强制对齐**：提取词级（Word-level）时间戳
    3.  **语义重组**：按 NLP 意群重组为完美语义句
    4.  **翻译与发声**：生成中文翻译及 TTS 音频并记录时长
    5.  **时间轴重构**：全局重算时间轴并生成 SRT 字幕
    6.  **视频组装**：原片切片 -> 提取尾帧画面定格 -> 混入 TTS -> 拼接
    7.  **字幕烧录与导出**：硬烧录字幕、剥离音频、清理临时文件

## 3. 核心功能与技术选型架构 (Architecture & Tech Stack)
* **3.1 资源获取模块**
    * **功能**：拉取最高质量音视频源文件
    * **选型**：`yt-dlp`
* **3.2 语音识别与语义重组模块 (核心算法层)**
    * **ASR & 强制对齐**：`WhisperX` (精确到毫秒的词级时间戳)
    * **语义断句 (Chunking)**：`spaCy` 等 NLP 工具或轻量级大模型
    * **时间映射**：反查首尾词时间戳获取重组后精准起止时间
* **3.3 文本处理与语音生成模块**
    * **翻译**：DeepL API / OpenAI API / 本地翻译模型
    * **TTS**：`edge-tts` (生成并精确计算音频时长)
* **3.4 视频剪辑与定格渲染引擎 (Video Engine)**
    * **选型**：FFmpeg / MoviePy
    * **核心逻辑**：截取原片 -> 抽尾帧 -> 生成匹配 TTS 时长的定格视频 -> 音视频合并 -> 并发处理 -> 极速无损拼接
* **3.5 字幕烧录模块 (Subtitle Burn-in)**
    * **选型**：FFmpeg `subtitles` 滤镜
    * **时间轴重算**：累加插入的 TTS 时长计算偏移量

## 4. 最终交付物 (Deliverables: Study Pack)
* **`[Title]_Shadowing_Bilingual.mp4`**：核心成品（双语、原音+定格配音、硬字幕）
* **`[Title]_Shadowing.mp3`**：纯音频版（适合通勤/盲听）
* **`[Title]_Bilingual.srt`**：时间轴重构后的外挂双语字幕
* **`[Title]_WordLevel_Data.json`**：原始词级时间戳与翻译语料（数据备份/后续扩展）

## 5. 性能与非功能需求 (NFR)
* **并行优化**：视频切片、静帧生成及 TTS 等过程需高度并行化
* **资源回收机制**：严谨的 `try...finally` 机制清理 `/temp` 目录碎片，防止空间泄漏
* **网络容错**：下载及 API 请求实现重试与超时机制（Retry Backoff）
